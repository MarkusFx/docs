Введение ​

Что такое Python? ​

Python - высокоуровневый интерпретируемый язык программирования.

Установка Python ​

Для работы с Python необходимо скачать и установить интерпретатор (программа, которая читает, анализирует и выполняет код построчно):

  • https://www.python.org/downloads/;

Правильность установки можно проверить выполнив команду в терминале:

shell
python --version

Первая программа ​

Например, создадим и изменим файл main.py:

main.py
py
print('Hello...')
1

В терминале выполним команду:

shell
python main.py
