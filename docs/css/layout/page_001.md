# Введение

## Что такое верстка?

**Верстка** - это процесс создания структуры и внешнего вида веб-страницы с помощью **HTML**, **CSS** (_и **JavaScript**_).

## Виды верстки

- **Блочная**;
- **Верстка на float** (_устаревшая_);
- **Табличная** (_устаревшая_);
- **Flexbox-верстка**;
- **Grid-верстка**;
- **Резиновая верстка**;
- **Адаптивная верстка**;
